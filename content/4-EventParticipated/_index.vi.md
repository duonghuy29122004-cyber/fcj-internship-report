# Sự kiện Tham gia

Trong thời gian thực tập, tôi đã tích cực tham gia các sự kiện và workshop của AWS để nâng cao kiến thức về cloud computing và giao lưu với các chuyên gia trong ngành.

---

## Tổng kết Sự kiện

### 1. AWS Cloud Mastery Series #1 - AI/ML/GenAI trên AWS
📅 **15 tháng 11, 2025** | 🕣 **8:30 AM – 12:00 PM**

**Chủ đề chính:**
- Nền tảng ML toàn diện Amazon SageMaker
- Generative AI với Amazon Bedrock
- Foundation Models (Claude, Llama, Titan)
- Kiến trúc RAG (Retrieval-Augmented Generation)

**Kỹ năng Thu được:** Triển khai ML model, prompt engineering, triển khai RAG

---

### 2. AWS Cloud Mastery Series #2 - DevOps trên AWS
📅 **17 tháng 11, 2025** | 🕣 **8:30 AM – 5:00 PM**

**Chủ đề chính:**
- CI/CD Pipeline với CodeCommit, CodeBuild, CodeDeploy, CodePipeline
- Infrastructure as Code (CloudFormation & CDK)
- Dịch vụ Container (ECS, EKS, ECR)
- Monitoring & Observability (CloudWatch, X-Ray)

**Kỹ năng Thu được:** Tự động hóa CI/CD, triển khai IaC, container orchestration

---

### 3. AWS Cloud Mastery Series #3 - Security Pillar
📅 **29 tháng 11, 2025** | 🕣 **8:30 AM – 12:00 PM**

- Dịch vụ bảo mật: GuardDuty, Security Hub, CloudTrail
- Mã hóa với KMS và Secrets Manager

**Kỹ năng Thu được:** Best practices bảo mật, thiết kế IAM policy, tự động hóa incident response
