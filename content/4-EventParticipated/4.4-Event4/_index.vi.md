# Workshop Phát triển Hướng AI
## Amazon Q Developer & Kiro IDE

### Thông tin Sự kiện

📅 **Thứ Sáu, 3 tháng 10, 2025**  
🕑 **2:00 PM – 4:30 PM**  
📍 **AWS Event Hall, L26 Bitexco Tower, HCMC**

**Giảng viên:** Toan Huynh & My Nguyen  
**Điều phối:** Diem My, Dai Truong, Dinh Nguyen

### Chương trình

#### 2:00 PM - 2:15 PM | Chào mừng

- Giới thiệu về AI-Driven Development
- Tổng quan workshop và mục tiêu
- Giới thiệu giảng viên và người tham gia

#### 2:15 PM - 3:30 PM | Vòng đời Phát triển Hướng AI

**Trình bày bởi: Toan Huynh**

- **Tổng quan AI-Driven Development Life Cycle**
  - Phát triển phần mềm hiện đại với sự hỗ trợ của AI
  - Lợi ích và tăng năng suất
  - Tích hợp vào quy trình làm việc hiện có

- **Demo Amazon Q Developer**
  - Tạo code và code completion
  - Giải thích code và tài liệu hóa
  - Phát hiện bug và quét bảo mật
  - Tạo test
  - Refactoring và tối ưu hóa code
  - Hỗ trợ đa ngôn ngữ
  - Demo coding trực tiếp

#### 3:30 PM - 3:45 PM | Giải lao

#### 3:45 PM - 4:30 PM | Demo Kiro IDE

**Trình bày bởi: My Nguyen**

- **Tổng quan Kiro IDE**
  - Môi trường phát triển được hỗ trợ bởi AI
  - Hỗ trợ code thông minh
  - Gợi ý nhận biết ngữ cảnh
  - Tích hợp AI chat và tạo code

- **Demo Trực tiếp**
  - Thiết lập Kiro IDE
  - Quy trình coding được hỗ trợ bởi AI
  - Các tình huống phát triển thực tế
  - Best practices và tips
